## üìã **What Are Data Brokers?**

Data brokers collect and sell personal information including:
- Full name, age, date of birth
- Current and past addresses
- Phone numbers and email addresses
- Family members and relatives
- Property ownership records
- Court records and criminal history
- Social media profiles
- Purchasing habits and interests

**Major data brokers**: Spokeo, Whitepages, BeenVerified, Intelius, PeopleFinder, MyLife, TruthFinder, Instant Checkmate, and 100+ others.

  - PeopleFinder.com

- [ ] **Document what information they have**
  - Screenshot each profile

### Phase 2: Major US Data Brokers (Week 1 - Manual)

- [ ] **WhitePages.com**
  - Submit removal request at whitepages.com/suppression_requests
  - Provide: Name, current address, phone number
  - Expected processing: 24-48 hours

- [ ] **Spokeo.com** 
  - Use privacy dashboard at spokeo.com/optout
  - Search for your profile first, then request removal
  - Expected processing: 72 hours

- [ ] **BeenVerified.com**
  - Email privacy request to privacy@beenverified.com
  - Include: Full name, all known addresses, phone numbers
  - Expected processing: 5-7 business days

- [ ] **TruePeopleSearch.com**
  - Submit suppression request at truepeoplesearch.com/removal
  - Requires phone verification
  - Expected processing: 48 hours

### Phase 2: Marketing Data Brokers (Week 1-2)

- [ ] **Acxiom Consumer Portal**
  - Create account at acxiom.com/about-acxiom/privacy/opt-out
  - Complete consumer data opt-out form
  - Expected processing: 6-8 weeks

- [ ] **LexisNexis Privacy Portal**
  - Submit request at optout.lexisnexis.com
  - Provide ID verification documents
  - Expected processing: 5-7 business days

- [ ] **Epsilon Marketing**
  - Email opt-out request to privacy@epsilon.com
  - Include all known email addresses and mailing addresses
  - Expected processing: 30 days

- [ ] **Experian Marketing Services**
  - Use consumer opt-out at experian.com/privacy/opt-out
  - Separate from credit report - this is marketing data
  - Expected processing: 30 days

### Phase 3: UK-Specific Data Brokers (Week 2)

- [ ] **192.com Directory**
  - Submit removal request at 192.com/about/privacy
  - UK address and phone number required
  - Expected processing: 28 days

- [ ] **BT Phonebook**
  - Call 0800 833 400 or online at bt.com/privacy
  - Request directory opt-out
  - Expected processing: Next directory publication

- [ ] **Electoral Roll Removal**
  - Contact local council electoral services
  - Request removal from open electoral register
  - Keep on full register for credit/legal purposes

### Phase 4: Follow-up & Verification (Week 3)

- [ ] **Verify removals completed**
  - Search for your name on each site after processing time
  - Document any sites that didn't process removal
  - Submit follow-up requests for non-compliance

- [ ] **Set up monitoring**
  - Google Alerts for your name and old email
  - Quarterly checks on major people search sites
  - Document new sites that appear

## üöß **Potential Issues & Solutions**

### Issue: Site requires ID verification
**Solution**: Use secure document upload, never email sensitive documents

### Issue: Removal request rejected or ignored
**Solution**: Follow up with GDPR/CCPA legal requests, document non-compliance

### Issue: Information reappears after removal
**Solution**: These sites often re-aggregate data, requires ongoing monitoring
